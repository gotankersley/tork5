Hotkeys and shortcuts for tork5

Tork5 has a couple different shorcuts to help with testing and debugging:

Ctrl + clickContinue pin placing
Double click pinSkips rotation (only when allowed)
Alt + clickContinue quad rotation
EnterChange turn
SpaceSwitch to place pin mode
DeleteRemove pin (place mode only)
FFind wins
RSwitch to rotate quad mode
SShow board score (in console)
